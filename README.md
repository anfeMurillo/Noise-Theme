# Noise ZX
Noise ZX is a dark material-like theme available in 14 vibrant color variants.

## Color Variants
- **Noise Red** - Classic red tones
- **Noise Blue** - Cool blue tones
- **Noise Green** - Fresh green tones
- **Noise Brown** - Warm brown/coffee tones
- **Noise Purple** - Vibrant purple/magenta tones
- **Noise Cyan** - Cyan/aquamarine tones
- **Noise Yellow** - Bright yellow tones
- **Noise Pink** - Pink/fuschia tones
- **Noise Orange** - Energetic orange tones
- **Noise Magenta** - Intense magenta tones
- **Noise Teal** - Teal/turquoise tones
- **Noise Indigo** - Deep indigo/blue-purple tones
- **Noise Lime** - Bright lime/yellow-green tones
- **Noise Coral** - Coral/orange-pink tones

## Installation
 1. Open Extensions on the sidebar
 2. Search for "**Noise ZX**"
 3. Install it
 4. Reload the editor
 5. Press ⌘K + ⌘T (or Ctrl+K + Ctrl+T on Windows/Linux)
 6. Pick your favorite color variant from the list and hit enter
 7. Enjoy!
