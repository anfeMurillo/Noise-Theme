# Noise ZX
Noise ZX is a dark material-like theme available in two color variants: red and blue tones.

## Installation
 1. Open Extensions on the sidebar
 2. Search for "**Noise ZX**"
 3. Install it
 4. Reload the editor
 5. Press ⌘K + ⌘T
 6. Pick "**Noise Red**" or "**Noise Blue**" from the list and hit enter
 7. Enjoy!
